Lexical behavior



Lexical behavior involves using specific words, terms, or verbs in a message to enhance the sender’s intention to mislead a receiver. This type consists of five specific behaviors.

1.
The first lexical behavior is using a first-person pronoun.

For example, you can think of a sentence like, “I’m pretty sure that this information can help you a lot.” In this sentence, “I” is a first-person pronoun of the lexical behavior.

2.
The second lexical behavior involves using interaction requesting words that request users to take an action with an online account or a link such as ‘update, confirm, verify, secure, log, click, inconvenient, and account.’

For instance, in the sentence “Click the button below to update your personal information”, the word "click" is used.

3.
The third lexical behavior involves using generalizing terms like ‘everyone, all, anybody.’

For example, in the sentence, “Everyone believes that our economy will improve next year”, a generalizing term "everyone" is used.

4.
The fourth lexical behavior involves using exclusive words such as ‘but, except, without.’

For example, in the sentence, “Benefit is clear without loss”, an exclusive word ‘without’ is used.
